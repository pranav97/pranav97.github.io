Algorithm Visualizer for Number of Islands

Problems can have many solutions. In the case of the Number of Islands problem there are at least 3 different solutions.

Problem Statement

Inputs: Grid of 0s and 1s. 1 indicates land and 0 indicates water.

Example:
Input:
11000
11000
00100
00011

Output: Count the number of 'Islands'. An island is a 1 or a land mass by water on all 4 sides. (up, down, left, right)

Example output: 3

Depth First Search

In the context of a graph, depth first search technique visits neighbors until it reaches the leaf nodes.

In the case of this grid, we are searching for land until we can't find land anymore.

Breadth-first search

In the context of a graph, breath first search technique visits closest neighbors to the source until it reaches the leaf nodes.

In the case of this grid, we are searching for land until we can't find land anymore.

Union Find or Disjoint-set

Union Find is a data structure that tracks a set of elements partitioned into disjoint or non-overlapping subsets. Many problems in the real world require algorithms that keep track of non-overlapping subsets efficiently. For example,

  • Finding out the number of connected components in a digital picture.
  • Finding out if there exists a path between two nodes in a network.
  • Kruskal's algorithm - This greedy algorithm finds the minimum weight spanning tree for a connected graph. A minimum weight spanning tree has all the vertices of the fully connected graph. More importantly, the algorithm minimizes the weights in the graph. In a network situation, nodes would prefer to route traffic through edges in the minimum weight spanning tree as opposed to edges that are not to minimize cost.
    • For more information about Kruskal's algorithm, click here.
  • For more information about the union find algorithm, click here.

In the case of this grid, all of the land masses or 1s are going to be elements in the disjoint-set data structure. After joining or unifying neighboring sets in the grid, the number of islands in the grid can be found out.
