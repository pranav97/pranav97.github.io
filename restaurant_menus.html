Restaurant Menus Website

This project has a collection of restaurants and menus in them kind of like GrubHub or Uber Eats. It is intended to give users the ability to look at the eat in menu before coming to the restaurant so they know what to order ahead of time.

It uses a sqlite database with ORM for database interactions and python flask for the web framework.

This is a flask project. Flask is micro web development framework that was created based on the WSGI(Web Server Gateway Interface) toolkit. WSGI is a simple calling convention that was created for webservers to forward requests to client side web apps written in Python.

One of the biggest take aways for me from this project is the basic web development concepts like kinds of HTTP verbs or request methods (get put post delete) and the response codes.

I also built my own authentication and authorization systems and realized why it's good to have third party authorization such as a google log in.

Each user gets to see a bunch of template restaurants and menus. After they log in they can add their own menus that will not be visible to other users. Users also can't delte the template menus and restaurants or other users content (because they can't see them).

I've outlined some of the cool features of this website in case you don't want to explore around yourself even though I would highly recomend it, sign up only takes 10s.
